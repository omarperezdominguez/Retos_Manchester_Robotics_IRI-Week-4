**Reto Manchester Robotics: Navegación Autónoma con Visión Computacional**

Este repositorio contiene la implementación de un sistema de navegación para un robot móvil diferencial desarrollado en ROS 2 Humble. El proyecto integra el control cinemático punto a punto con una capa de toma de decisiones basada en visión artificial para la detección de semáforos en tiempo real, ejecutado en una plataforma NVIDIA Jetson Nano.
